# Personal Dotfiles.

I am currently using arch linux + hyprland. my config is based in the caelestia dotfiles [caelestia dotfiles](https://github.com/caelestia-dots/caelestia), but with some modifications to fit my needs.

This setup is made to be used by me, **so it may not fullfill all your needs**, but, if you want to use it: feel free to do so,

## Installation

```bash
git clone https://github.com/jskahl/dotfiles.git $HOME/dotfiles
cd $HOME/dotfiles
chmod +x install.sh
./install.sh
```
